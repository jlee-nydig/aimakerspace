Context Recall

Measures how much of the ground-truth reference is recoverable from the retrieved context.

Input 1
User Query
Input 2
Retrieved Context
Input 3
Ground-Truth Reference
1
Decompose
Break the reference into individual claims
An LLM reads the ground-truth reference and splits it into atomic, standalone factual claims.
Example — Reference "Albert Einstein developed the theory of relativity. He was born in Ulm, Germany in 1879."

→ Claims extracted
① Einstein developed the theory of relativity
② Einstein was born in Ulm, Germany
③ Einstein was born in 1879
2
Verify
Check each claim against the retrieved context
For every claim, an LLM judges: "Can this claim be attributed to (supported by) the retrieved context?"
Retrieved Context "Einstein is famous for the theory of relativity, published in 1905 and 1915. He was born in 1879."

→ Verification
✓
Einstein developed the theory of relativity
✗
Einstein was born in Ulm, Germany — context doesn't mention Ulm
✓
Einstein was born in 1879
3
Count
Tally supported vs. total claims
Count how many claims were supported by the context and how many total claims were extracted from the reference.
Tally
Supported claims:  2
Total claims:  3
4
Compute
Calculate the Context Recall score
Divide supported claims by total claims. The result is a value between 0 (none of the reference is recoverable) and 1 (all of it is).
Context Recall =
claims in reference supported by context total claims in reference
